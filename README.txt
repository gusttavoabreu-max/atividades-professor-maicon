======================================
  AGROSIM BRASIL 2.0
  Simulador de Trator 3D
======================================

COMO JOGAR:
-----------
1. Abra o arquivo "index.html" no navegador
   (Chrome ou Firefox recomendado)

2. Clique na tela para travar o mouse

CONTROLES:
----------
WASD          - Mover personagem / dirigir veículo
Mouse         - Câmera (clique na tela primeiro)
E             - Entrar no veículo (perto dele) / Sair do veículo
R             - Ligar / desligar motor
ESPAÇO        - Ativar corte da colheitadeira
F             - Alternar câmera 1ª / 3ª pessoa

VEÍCULOS:
---------
🚜 Trator John Deere 6M   - canto direito do mapa
🌾 Colheitadeira New Holland CR - esquerda

DICAS:
------
- Entre na Colheitadeira, ligue o motor (R),
  depois pressione ESPAÇO para ativar o corte
- Passe pelos campos de plantação para colher
- Cada colheita dá dinheiro!
